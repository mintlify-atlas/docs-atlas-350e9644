## How it works

<Steps>
  <Step title="Install packages">
    Add `vite-plus`, `eslint`, `stylelint`, and the Angular ESLint plugins to your devDependencies.
  </Step>
  <Step title="Add config files">
    Copy `.oxlintrc.json`, `stylelintrc.json`, `eslint.config.ts`, `.oxfmtrc.json`, and `vite.config.ts` into your project root.
  </Step>
  <Step title="Add npm scripts and install">
    Add the lint, format, and `postinstall` scripts to `package.json`, then run `pnpm install` to install packages and register the Git hook automatically.
  </Step>
  <Step title="Commit with confidence">
    Every `git commit` triggers the pre-commit hook, which checks only your staged files and blocks the commit if any unfixable errors are found.
  </Step>
</Steps>
